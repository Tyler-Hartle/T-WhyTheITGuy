# T-Why Page Engagement Strategy I rebuilt the page around child-friendly engagement patterns rather than a plain landing page. ## Layout strategy - **Character first:** T-Why appears immediately in the hero because kids connect faster with a guide/mascot than with a block of text. - **One big promise:** The hero repeats one rule: “Pause. Ask why. Ask a grown-up.” Repetition helps retention. - **Short visual chunks:** The page avoids long paragraphs and uses cards, icons, XP, badges, and color blocks. - **Immediate action:** The first calls to action are Watch and Pick a Mission, not “learn more.” - **Game loop:** The Safety Quest uses a simple loop: question → choice → feedback → XP → badge. - **Repeatable content engine:** Each mission maps directly into an episode, printable lesson, game level, and social post. ## Metrics to track after launch - 3-second video retention - 10-second video retention - Full video completion rate - Quest start rate - Quest completion rate - Badge unlock rate - Repeat visits - Parent/teacher resource clicks - YouTube/Shorts subscribe conversion ## Recommended content cadence Start with 8 short episodes. Launch two per week for four weeks, then review which topics get the best completion and engagement before scaling. ## Important note For a kid-facing brand, keep the current page privacy-first: no accounts, no personal data forms, no chat, no user uploads, and no open comments on the website itself.
